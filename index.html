Action Classification in Still Images Using Human Gaze
Under the guidance of Kiwon Yun, Prof. Dimitris Samaras and Prof. Greg Zelinsky at Stony Brook University's Image Analysis Lab, I researched the application of human gaze data to computer vision algorithms for recognizing actions in still images. Using gaze data collected from 12 subjects with a tower-mounted eye tracker, I computed novel gaze features that, when used to train support vector machine classifiers, improved on those trained with state-of-the-art visual features. By blurring the border between psychology and computer vision, I uncovered new insights into the way humans interpret images while opening a new direction for the advancement of image analysis algorithms.
